# WAD2024 node server branch


## Pre-requisites
Node.js \& NPM

## Clone the repository
```
git clone https://github.com/Jawa4a/WAD2024/tree/server
```
or
```
git clone https://github.com/Jawa4a/WAD2024
```
and 
```
git checkout server
```

## Navigate to the project directory
```
cd <project name>
```

## Install dependencies
```
npm install
```

## Run the project
```
node server
```

or 
```
nodemon server
```
